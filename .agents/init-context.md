# Init Context

Read the project AGENTS.md files to understand context before starting work.

Steps:
1. Read `AGENTS.md` in the project root for overview, stack, and structure
2. Read `packages/*/AGENTS.md` for the package(s) relevant to the user's request
3. Summarize: project purpose, stack, monorepo structure, and 1-2 pending items

Keep under 150 words. End with: "Ready to help — what would you like to work on?"
